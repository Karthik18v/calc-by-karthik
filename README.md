## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Karthik18v/calc-by-karthik.git
    ```

2. Navigate to the project directory:
    ```bash
    cd MyExpoApp
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Start the project:
    ```bash
    npm start
    ```
